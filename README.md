# Specialized Serviços - Site Institucional

Este repositório contém o código-fonte do site institucional da Specialized Serviços, uma empresa focada em terceirização de serviços para condomínios e empresas no Distrito Federal. O projeto foi desenvolvido para apresentar a marca, detalhar os serviços oferecidos e facilitar o contato com novos clientes através de uma interface moderna e responsiva.

## Visão Geral do Projeto

O objetivo principal deste projeto é fornecer uma presença digital sólida para a empresa, destacando seus diferenciais competitivos, como gestão de facilidades, portaria, limpeza e manutenção predial. O site foi otimizado para desempenho e acessibilidade, garantindo uma navegação fluida em dispositivos móveis e desktops.

### Funcionalidades Principais

* **Apresentação Institucional:** Seções dedicadas à história, missão e valores da empresa.
* **Catálogo de Serviços:** Detalhamento das áreas de atuação (Portaria, Limpeza, Jardinagem, Manutenção).
* **Galeria de Projetos:** Exposição visual de trabalhos realizados e equipes em campo.
* **Integração de Contato:** Botões de ação direta para WhatsApp e formulários de solicitação de orçamento.
* **Design Responsivo:** Layout adaptável para smartphones, tablets e computadores.

## Tecnologias Utilizadas

O desenvolvimento priorizou o desempenho e a manutenção simplificada, utilizando tecnologias web padrão:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização personalizada e design responsivo.
* **JavaScript:** Interatividade e manipulação do DOM para melhor experiência do usuário.

## Instalação e Execução Local

Para visualizar e editar este projeto em sua máquina local, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/ObarbosaDev/SpecializedCondominios-Site.git](https://github.com/ObarbosaDev/SpecializedCondominios-Site.git)
    ```

2.  **Acesse o diretório do projeto:**
    ```bash
    cd SpecializedCondominios-Site
    ```

3.  **Execute o projeto:**
    * Como se trata de um projeto estático, você pode abrir o arquivo `index.html` diretamente em seu navegador.
    * Para uma experiência de desenvolvimento mais robusta, recomenda-se o uso de uma extensão como o "Live Server" no VS Code.

## Deploy

O projeto está atualmente hospedado e ativo na plataforma Netlify. A integração contínua (CI/CD) pode ser configurada para atualizar automaticamente o site a cada novo commit na branch principal.
